Aprendendo as Tags utilizadas em Html5

Parágrafos e suas formatações

Inicialmente para criar-mos um paragrafo em html utilizamos as tags <p>antes do conteúdo e </p> para finalizá-lo

Negrito

Para aplicar a formatação Negrito, escrevemos o texto a ser formatado entre as tags <strong> e </strong>

Italico

Para aplicar a formatação Itálico, escrevemos o texto a ser formatado entre as tags <i> e </i>

Quebra de linha

Em html para se fazer uma quebra de linha
Utilizamos a tag <br>
esta, não possui tag de fechamento

Sublinhado

Para aplicar a formatação Sublinhado, escrevemos o texto a ser formatado entre as tags <u> e </u>

Sobrescrito

Para aplicar a formatação Sobrescrito, escrevemos o texto a ser formatado entre as tags <sup> e </sup>

Maior

Para aplicar a formatação aumentando o tamanho da letra, utilizava-se as tags <big> e </big> até o html4. apesar dessas tags ainda estarem funcionando, no html5 convencionou-se aumetar a letra utilizando a estilização "font-size" no css3

Menor

Para aplicar a formatação reduzindo o tamanho da letra, escrevemos o texto a ser formatado entre as tags <small> e </small>.


Linha Horizontal

Para se criar uma linha horizontal, utiliza-se a tag <hr>. Esta, não possui tag de fechamento

Estilos

Para aplicar a estilização ao html, normalmente utiliza-se o css3. Porém outra forma seria utilizando a tag <style> dentro do <head>. Assim como fiz para alterar a cor do texto deste paragrafo.


Listas

Para se criar uma em html utliza-se as tags <ul> para listas desordenadas e <ol> para listas ordenadas
