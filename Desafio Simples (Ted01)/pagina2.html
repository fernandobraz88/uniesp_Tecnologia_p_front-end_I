Como criar botões e direcionálos para outra Página.

Botões: Para criar botões a tag utilizada é <button>
Link entre Páginas: Para criar um link entre páginas a tag é <a> preenchendo o href= com a pagina destino

Então para criar um botão que faz um link para outra página, utilizamos a tag <a> dentro da tag <button> como no exemplo abaixo:
<button><a href="Página destino"> Texto do Botão </a></button>

Assim fiz nos botõs "Home" e "Pagina 2" na Página anterior.
